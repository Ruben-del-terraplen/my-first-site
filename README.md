# rubencurbelo.com

Personal portfolio site for Ruben Curbelo, Senior UX/UI Designer.

**Status:** Phase 1 — design preview deployed. Real content swap coming next.

## Stack (current)

Static HTML/CSS/JS. Design generated by Claude Design. Deployed on Vercel.

## Stack (planned)

Will be migrated to Next.js + TypeScript + Tailwind by Claude Code, with content from `CONTENT_PLAN.md` and assets from the `framer-assets/` folder.

## Live

https://my-first-site-rho-one.vercel.app

## Local preview

No build step needed. Open `index.html` in a browser, or run a tiny local server:

```
npx serve .
```

## Files

- `index.html` — homepage
- `case-1-casino.html` ... `case-4-niftykit.html` — case study pages
- `styles.css` — shared homepage styles
- `case-study.css` — shared case study styles
- `reveal.js` — small scroll-reveal animation script
- `vercel.json` — Vercel static-hosting config
